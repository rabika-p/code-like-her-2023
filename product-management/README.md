# Product Management

## Installation
1. Clone the repository:  
   `git clone https://github.com/rabika-p/code-like-her-2023.git`  
   `cd code-like-her-2023/product-management`

2. Install the dependencies:  
    `npm install`  
   
## Usage  
To start the app in development mode, run:  
    `npm run start`
